# BAD MOUTH - Movies & Music Recommendation App

Swipe right on movies and music you love. Get personalized recommendations based on your mood.

## Tech Stack
- React + TypeScript (Vite)
- Supabase (Auth + Database)
- Vercel (Hosting)

## Development
```bash
cd frontend
npm install
npm run dev
```

## Features
- 🎬 Movie recommendations from TMDB
- 🎵 Music recommendations from Spotify
- 👆 Tinder-style swiping interface
- 📋 Watch/Listen list
- 🎭 Mood-based discovery
- 👑 Admin dashboard
